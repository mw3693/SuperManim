الاقتراحات في هذا الملف مجرد اقتراحات لتصميم حالة المشروع، وإذا كان فيها أخطاء يمكن تصحيحها. الهدف هو تصميم أداة قوية وذكية. استخدم هذه الاقتراحات في تخطيط حالة المشروع، واكتب لي Module 10 بالتفصيل الممل جدًا، وباللغة الإنجليزية البسيطة جدًا، ويكون الشرح طويل، مع رسومات توضح كل الحالات وكل الـ cases.
كذلك، خذ في الاعتبار تغطية كل الحالات، بما في ذلك إذا الأداة أغلقت بسبب انقطاع الكهرباء أو إذا المستخدم أغلقها بطريقة خاطئة (not cleanly).

انا عاوز اعمل حاجتين وهى حالة المشروع ودي فايدتها لو فتحت مشروع كنت شغال عليه قبل كدة الأداة تقوم باسترجاع كل شيء عن المشروع ده إعداداته كلها وكمان اعرف استرجع اخر مشروع كان مفتوح last opened project وكمان اقدر اعمل list recent projects

فأنا قلت اعمل التالى هعمل entity اسمها projectState دي اخزن فيها كل حاجة عن المشروع ده أثناء فترة تشغيل الأداة بيه

يعنى لو فاتح الأداة وشغال على مشروع ما بعمل فيديو مثلا فإن projectState ده كيان فى المشروع وكمان له جدول فى قاعدة البيانات اسمه project_state بيتخزن فيه كل حاجة عن حالة المشروع فى وقت التشغيل

وهيكون فيه adaptor اسمه project_state_repository.py وهيكون ليه بورت projectStateRepositoryPort

وفيه خدمة كده application service فايدتها فقط هى انها تقوم بتحديث الجدول ده فى قاعدة البيانات طول فترة تشغيل الأداة والعمل على المشروع وهيكون فيها الدوال التالية save_state restore_state بتاخد بيانات الحالة دي وتملا بيها باقي قواعد البيانات بالظبط ده لو هسترجع المشروع من جديد طبعا لازم اتأكد الأول إن الملف project_data.db موجود والجدول بتاع حالة المشروع

طول فترة التشغيل الخدمة دي تقوم بتحديث حالة المشروع كله ككل مفيش حد يقدر يعدل على الجدول ده إلا الـ adaptor project_state_repository adapter





انا عاوز اول ما الأداة تفتح أول مرة خالص أول حاجة تحصل إن خدمة projectStateService تشوف فيه مشروع مفتوح ولا لا، لو فيه مشروع مفتوح تمام تحفظ الحالة بتاعته فى الذاكرة، لو مفيش مشروع فى الحالة دي تنتظر المستخدم إنه يفتح مشروع open project أو يقترح على المستخدم إنه يفتح أو يسترجع آخر مشروع كان مفتوح، فتروح تقرأ ملف session.db وتشوف

المشروع له حالتين الحالة الأولى STATE 1 ودي الحالة اللى فيها المشروع مش مفتوح، الأداة مفتوحة ولكن مفيش مشروع مفتوح

الحالة الثانية STATE 2 الحالة اللى فيها مشروع مفتوح والأداة فاتحة

فيه كذا حالة الخدمة SERVICE PROJECT STATE هتروح تقرأ ملف SESSION

الحالة الأولى إنها ملقتش ملف أصلاً، فى الحالة دي هيبقى الأداة فى الحالة رقم 1 والـ is_project_open بيساوي صفر أو false، وتشتغل البرامبت بتاع الأداة وتنتظر المستخدم إنه يفتح مشروع جديد أو ينشئ واحد

الحالة الثانية لقت ملف session ولكنه فارغ مفيهوش حاجة، الحالة دي المستخدم كان قافل الأداة والمشروع cleanly، برضه هنخلي الأداة على الحالة 1 والفلاج هنخليه false is_project_open

الحالة الثالثة الملف مش فارغ ولكن فيه بيانات، فنقرأ البيانات last_opened_project_path نتأكد أولاً إنها بتشير لمشروع صالح، لو مكنتش بتشير لمشروع صالح يبقى فى الحالة دي الأداة مش هتعرف تسترجع حاجة، وعليه هنرجعها للوضع الأول STATE 1 والفلاج هيكون false

الحالة الرابعة المسار جوه الـ session بيشير لملف مشروع صالح وكمان ملفات المشروع صح، البيانات صح كله موجود، فى الحالة دي نظهر للمستخدم رسالة ونقوله فيه مشروع كان مفتوح، تحب تفتحه؟ اضغط yes، لو لا اضغط no، واستخدم الأوامر دي لعمل مشروع جديد أو فتح مشروع معين بمسار أنت عارفه أو list recent projects

CREATE TABLE project_state (
    -- ═══════════════════════════════════════════════════════════════
    -- SECTION 1: Project Identity (من القراءة فقط - من project_settings)
    -- ═══════════════════════════════════════════════════════════════
    project_id              INTEGER PRIMARY KEY DEFAULT 1,
    project_name            TEXT    NOT NULL,
    project_path            TEXT    NOT NULL,
    project_mode            TEXT    NOT NULL,  -- normal/simplemanim/supermanim
    
    -- ═══════════════════════════════════════════════════════════════
    -- SECTION 2: Scene Statistics
    -- ═══════════════════════════════════════════════════════════════
    total_scenes            INTEGER DEFAULT 0,
    rendered_scenes         INTEGER DEFAULT 0,
    pending_scenes          INTEGER DEFAULT 0,
    failed_scenes           INTEGER DEFAULT 0,
    skipped_scenes          INTEGER DEFAULT 0,
    
    -- ═══════════════════════════════════════════════════════════════
    -- SECTION 3: Audio Status
    -- ═══════════════════════════════════════════════════════════════
    has_audio               INTEGER DEFAULT 0,        -- 0=false, 1=true
    audio_file_path         TEXT    DEFAULT NULL,
    audio_file_duration     REAL    DEFAULT NULL,     -- بالثواني
    audio_clips_count       INTEGER DEFAULT 0,        -- عدد المقاطع المقسمة
    synced_scenes           INTEGER DEFAULT 0,        -- scenes متزامنة مع audio
    
    -- ═══════════════════════════════════════════════════════════════
    -- SECTION 4: Timeline Status
    -- ═══════════════════════════════════════════════════════════════
    total_duration          REAL    DEFAULT 0.0,      -- مجموع durations
    timeline_valid          INTEGER DEFAULT 0,        -- هل timeline صحيح؟
    timeline_gap            REAL    DEFAULT NULL,     -- لو فيه gap، قد إيه؟
    
    -- ═══════════════════════════════════════════════════════════════
    -- SECTION 5: Render Status
    -- ═══════════════════════════════════════════════════════════════
    last_render_at          TEXT    DEFAULT NULL,     -- timestamp
    last_render_duration    REAL    DEFAULT NULL,     -- كم استغرقت؟
    render_progress         REAL    DEFAULT 0.0,      -- 0.0 to 1.0 (نسبة مئوية)
    has_render_errors       INTEGER DEFAULT 0,
    last_error_scene        INTEGER DEFAULT NULL,     -- scene_id اللي فيها error
    
    -- ═══════════════════════════════════════════════════════════════
    -- SECTION 6: Preview Status
    -- ═══════════════════════════════════════════════════════════════
    preview_scenes          INTEGER DEFAULT 0,        -- scenes اللي ليها preview
    last_preview_at         TEXT    DEFAULT NULL,
    
    -- ═══════════════════════════════════════════════════════════════
    -- SECTION 7: Export Status
    -- ═══════════════════════════════════════════════════════════════
    export_ready            INTEGER DEFAULT 0,        -- هل جاهز للتصدير؟
    last_export_at          TEXT    DEFAULT NULL,
    last_export_path        TEXT    DEFAULT NULL,     -- مسار آخر فيديو
    export_count            INTEGER DEFAULT 0,        -- عدد مرات التصدير
    
    -- ═══════════════════════════════════════════════════════════════
    -- SECTION 8: Session Info
    -- ═══════════════════════════════════════════════════════════════
    opened_count            INTEGER DEFAULT 1,        -- كم مرة اتفتح
    total_work_time         REAL    DEFAULT 0.0,      -- إجمالي وقت العمل (ثواني)
    
    -- ═══════════════════════════════════════════════════════════════
    -- SECTION 9: Timestamps
    -- ═══════════════════════════════════════════════════════════════
    state_created_at        TEXT    NOT NULL,
    state_updated_at        TEXT    NOT NULL
);
